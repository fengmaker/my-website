# ⚡️ 能量与作息标准协议 (Energy Protocols)

> "你无法管理时间，你只能管理能量。"

本协议旨在修复 **系统关机失败 (晚睡)** 和 **冷启动困难 (赖床)** 的核心 Bug。

---

## 🌙 协议 A：强制关机程序 (Shutdown Ritual)

**目标：** 解决入睡拖延、缓解睡前对未来的焦虑。
**触发时间：** `23:00` (硬性中断)

### 🚫 核心红线 (The Kill Switch)

!!! danger "物理隔离 (执行权重: 最高)"
    当时间到达 23:00：
    1.  **充电器必须插在卧室门外**（或必须下床才能拿到的位置）。
    2.  手机必须离开手边。
    
    *这是系统的底线。如果这一步没做，后面所有优化都无效。*

### 📋 执行步骤

1.  **焦虑卸载 (Brain Dump)**
    * *动作*：在床头纸质本子上写下让大脑空转的内容。
    * *模板*：
        * [ ] 明天最重要的一件事是：_________
        * [ ] 此刻让我焦虑的是：_________ （已存档，明天处理）

2.  **感官降噪**
    * [ ] 关掉大灯，只留床头灯。
    * [ ] 戴上耳塞（如果环境嘈杂）。

3.  **被动输入模式**
    * *允许动作*：阅读纸质书 / Kindle / 听英语播客（定书面模式）。
    * *禁止动作*：**严禁**打开微信、B站、知乎。

!!! success "关机成功标准"
    在 23:45 前闭上眼睛，且手机不在伸手可及的范围内。

## ☀️ 协议 B：系统冷启动程序 (Cold Start)

**目标：** 解决赖床、醒后无意识刷手机、大脑混沌。
**触发时间：** `7:30` (闹钟响)

### 🚫 核心红线 (The Firewall)

!!! failure "绝对禁止"
    在 **双脚离地** 且 **喝完水** 之前，**严禁** 解锁手机屏幕。
    *一旦醒来第一眼看的是短视频，今日多巴胺系统即刻崩溃。*

### 📋 执行步骤

1.  **物理唤醒 (Physical Trigger)**
    * *动作*：闹钟必须放在**需要下床**才能关掉的地方。
    * *心态*：不要思考“要不要起”，像机器一样去按掉闹钟。

2.  **液压注入 (Hydration)**
    * *动作*：一口气喝掉昨晚准备好的 **300ml 温水**。
    * *原理*：水进入身体会强制唤醒内脏，解除“睡眠模式”。

3.  **环境扫描 (Light)**
    * *动作*：立刻拉开窗帘（见光）或者直接走进洗手间洗脸。

4.  **微任务启动 (Micro-Task)**
    * *动作*：执行昨晚写下的“第一件小事”（如：整理被子）。
    * *时长*：2 分钟。

!!! tip "防回滚机制"
    如果喝完水还是想睡回笼觉？
    **SOP：** 允许睡，但必须把窗帘拉开，并不许盖被子。通常你会发现你睡不着了。
